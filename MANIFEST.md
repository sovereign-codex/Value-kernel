# VALUE KERNEL MANIFEST v1.0.0

This manifest defines the complete and canonical file set
for Value Kernel v1.0.0.

**Status:** Binding  
**Scope:** All compliant distributions and mirrors

---

## 1. Canonical File List (Binding)

A valid Value Kernel v1.0.0 distribution **MUST** contain
all of the following files, unmodified:

- `README.md`
- `VALUE_KERNEL.md`
- `CONTRACT.md`
- `COMPLIANCE.md`
- `GOVERNANCE.md`
- `VERIFY.md`
- `MANIFEST.md`
- `decision_examples.md`
- `decision_tests.schema.json`
- `decision_tests.json`
- `LICENSE`

No additional files are required.
No listed file may be omitted.

---

## 2. Completeness Requirement (Binding)

If any listed file is missing, altered, or replaced,
the distribution **MUST** be considered incomplete and non-compliant.

---

## 3. Distribution Rules (Binding)

- Files may be mirrored or redistributed only as a complete set.
- Partial distributions **MUST NOT** claim compliance.
- Packaging format (zip, tar, etc.) does not alter requirements.
